## Navigation

## Screens

## Admin Dashboard
- Cards: total members, active members today, total courses
- Tables: top 5 courses today, active members today (paged)

## Courses & Tasks
- Member can create course (auto-enroll creator)
- Double-click course -> tasks
- Double-click task -> submit completion (today)

## Notifications
- ADMIN can create notifications
- Members read + mark as read
